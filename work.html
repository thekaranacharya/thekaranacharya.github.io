My Projects

Vehicle detection & Classification on Indian streets

Developed an object detection & classification model to detect vehicles on the crowded streets of India and classify them into the following types -
Car, Bus, Truck, Two-wheeler & Three-wheeler.

Tools & Frameworks used : TensorFlow, ImageAI, LabelImg

Result:

The model is able to detect and classify vehicles from both - images & videos with a Mean Detection Probability - 90%!
Following is the output on an image never seen by the model -



As you can see, in such a crowded street, the model is able to detect a truck (even just the back of one), a car, and multiple two-wheelers and rickshaws (with some probabilities over 96%!)

Novel Poetry Generation using NLP

Generating a novel Irish poem using a set of Long Short-Term Memory cells (Recurrent Neural Networks).
Tools & Frameworks used : TensorFlow-Keras

Result:


"Then came a sailor" was the only set of words I provided to the trained model as a starting line. The next 100 words have been generated solely by the network!
P.S. If you look closely, there are very less repetitions in the predicted words.
It's quite common to see a large number of repeated words in an NLP model prediction. But here, some hyperparameters were tuned to minimise the number of repetitions.

Tools & Frameworks used : TensorFlow-Keras

Result:

The following result is achieved with the model with the set of layers -
Embedding - Dropout - Conv1D - MaxPooling1D - Bidirectional LSTM - 2 Dense layers.


As you can see here, there is NOT a steep increase in the validation loss.
This means, the confidence in prediction on the validation set is NOT decreasing over the course of the epochs.
